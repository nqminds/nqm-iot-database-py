getAggregateData(pipeline, filter={})[source]¶

Performs an aggregate query on the given dataset resource.

Please note that as MongoSQL is used, some queries may not work.

Parameters:
  • pipeline (Mapping[str, Any]) – The aggregate pipeline, as defined in [the mongodb docs](https://docs.mongodb.com/manual/core/aggregation-pipeline/). Can be given as a JSON object or as a stringified JSON object.
  • filter (Mapping[str, Any]) – A mongodb filter object. If omitted, all data will have the aggregation run on them.
Return type:

DatasetData
Returns:

The data made from the aggregrate query. This will have only one row of data, and will be in SQL types.

Example

>>> from nqm.iotdatabase.database import Database
>>> db = Database("", "memory", "w+")
>>> id = db.createDatabase(schema={"dataSchema": {"a": []}})
>>> db.addData({"a": x} for x in range(3)) == {"count": 3}
True
>>> datasetData = db.getAggregateData(
...     pipeline={"answer": {"$sum": "a"}},
...     filter={"a": {"$lte": 2}})
>>> datasetData.data == [
...     {"answer": sum(a for a in range(3) if a <= 2)}] # 1 + 2
True
getData(filter={}, projection={}, options={})[source]¶

Gets all data from the given dataset that matches the filter.

Parameters:
  • filter (Mapping[str, Any]) – A mongodb filter object. If omitted, all data will be retrieved.
  • projection (Mapping[str, int]) – A mongodb projection object. Should be used to restrict the payload to the minimum properties needed if a lot of data is being retrieved.
  • options (Mapping[str, Any]) – A mongodb options object. Can be used to limit, skip, sort etc.
Return type:

DatasetData
Returns:

An object containing the data retrieved in the data field.
Raises:
  • ValueError – If an invalid option is given.
  • TypeError – If the sort option is a dict, not an OrderedDict.

Example

>>> from nqm.iotdatabase.database import Database
>>> db = Database("", "memory", "w+")
>>> id = db.createDatabase(schema={"dataSchema": {"a": []}})
>>> db.addData([{"a": 1}, {"a": 2}]) == {"count": 2}
True
>>> datasetData = db.getData(filter={"a": 2})
>>> datasetData.data == [{"a": 2}]
True
getDataCount(filter={})[source]¶

Gets a count of the data that matches the filter.

Essentially just wraps getAggregateData() with args pipeline={"count": {"$sum": 1}}.

Parameters:filter (Mapping[str, Any]) – An optional mongodb filter to apply before counting the data.
Return type:DatasetCount
Returns:The metadata and count in the count field.

Example

>>> from nqm.iotdatabase.database import Database
>>> db = Database("", "memory", "w+")
>>> id = db.createDatabase(schema={"dataSchema": {"a": []}})
>>> db.addData({"a": x} for x in range(3)) == {"count": 3}
True
>>> datasetCount = db.getDataCount({"a": {"$lte": 2}})
>>> datasetCount.count == sum(1 for a in range(3) if a <= 2)
True
getResource()[source]¶

Gets the details/metadata for this dataset

Return type:MetaData

>>> from nqm.iotdatabase.database import Database

...     "schema": {"dataSchema": {"a": []}}, "name": "Hi World"}

Hi World
